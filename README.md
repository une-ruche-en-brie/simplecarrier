# Simple Carrier

SimpleCarrier is a module for [PrestaShop](https://prestashop.com/).

[![Phyxo](https://github.com/une-ruche-en-brie/simplecarrier/actions/workflows/php.yml/badge.svg)](https://github.com/une-ruche-en-brie/simplecarrier/actions/workflows/php.yml)


This module aims to provide a simple way to manage delivery using [Mondial Relay](https://www.mondialrelay.fr/).

The main developement has been by Mondial Relay(offrestart@mondialrelay.fr) under [OSL 3.0 license](http://opensource.org/licenses/osl-3.0.php).

# Installation

You can install that module from your PrestaShop administration area.

## Static code analysis

Analysis is made using [PHPStan](https://github.com/phpstan/phpstan) :

```
$ composer phpstan
```

The analysis is made with level 1 but the idea is to increase that level and fix more and more possible issues.
